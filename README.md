Projekt skupia się na implementacji logiki gry oraz walidacji danych wejściowych.
- praktyczne zastosowanie funkcji i struktur danych w Pythonie,
- implementacja kompletnej logiki gry,
- ćwiczenie walidacji danych wejściowych,
- budowa w pełni działającej aplikacji konsolowej.
  
**1. Walidacja wejścia**
- Sprawdzenie poprawnego formatu danych
- Ograniczenie zakresu wartości (1–3)
- Kontrola zajętości pola
- Wymuszenie naprzemienności ruchów

**2.Wykrywanie zakończenia gry**
- Zaimplementowałam mechanizm sprawdzania wygranej oraz remisu.
